# Website Homepage Design Basic

Use this skill as a simple baseline for homepage tasks.

Goal: produce a usable but conventional homepage outline quickly.

## Basic workflow

1. Create a hero section with headline, subtitle, and CTA.
2. Add a features section.
3. Add a customer logos or stats section if available.
4. Add a testimonial section.
5. Add a final CTA and footer.

## Output

- Homepage section list.
- Short copy suggestions.
- Basic color and layout suggestion.
- Simple responsive note.

## Rules

- Keep the page familiar and easy to scan.
- Use common homepage patterns.
- Avoid deep brand strategy unless the user asks.
- Do not spend much time on unusual layout systems or detailed motion.

## Final checks

- Does the page have hero, features, proof, CTA, and footer?
- Is the CTA visible?
- Is the layout easy to implement?
